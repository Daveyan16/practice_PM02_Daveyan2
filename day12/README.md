# Отладка программы - Вариант №6

## Описание проекта
Проект демонстрирует процесс отладки Python-программы с использованием специализированных средств отладки.

## Структура проекта
- `variant_6_original.py` - Исходный код с ошибками
- `variant_6_fixed.py` - Исправленный код
- `test_variant_6.py` - Юнит-тесты
- `.pdbrc` - Конфигурация pdb
- `reports/` - Отчеты и анализ

## Выявленные ошибки
1. Неправильная передача *args, **kwargs в декораторе
2. Логические ошибки в формулах
3. Утечки памяти в кешах
4. Ошибки состояния в декораторе-классе

## Запуск
```bash
# Запуск исходного кода с ошибками
python variant_6_original.py

# Запуск исправленной версии
python variant_6_fixed.py

# Запуск тестов
python -m pytest test_variant_6.py -v

# Запуск под отладчиком
python -m pdb variant_6_original.py
